# CSTP Access

Self-service access requests for the private **CSTP Module 1** course content
(Infopercept team only).

## How to get access

1. Sign in to GitHub (create a free account if you don't have one).
2. [Open an access request](https://github.com/divyesh-vaishnav-im/CSTP-Access/issues/new?template=request-access.yml)
   and enter your **work email**.
3. A 6-digit verification code is emailed to that address within 1–2 minutes.
4. Reply to your request with the code.
5. You'll receive a GitHub invitation to the course repository — accept it and you're in.

Access is granted automatically, but **only** for verified work email addresses
on an approved domain (see [`config/allowed-domains.json`](config/allowed-domains.json)).
Requests from any other domain are closed automatically.

> This repository contains no course material — it only handles access requests.
